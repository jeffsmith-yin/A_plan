## 二、分场景触达话术

### 2.1 脉脉私信 — 初次触达

```
[称呼]老师您好，

冒昧打扰。看到您在[行业]领域有[XX年]的经验，之前在[公司/职位]的
经历非常令人敬佩。

我目前在做一个创业项目——"融智桥"，专门帮资深行业专家和AI技术
人才搭班子，为中小微企业提供AI解决方案。

简单说：您出经验判断，AI人才出技术实现。平台负责匹配、签约、
用智能合约自动分账。您不需要学AI，只需要告诉AI人才"这个行业
怎么干"。

目前我们在招募首批种子专家（5-10人），参与DEMO验证。想邀请您
花3分钟填一份问卷，帮我们验证这个模式是否成立：
[专家端问卷链接]

如果合适，后续可以深度聊聊。您的经验在AI时代反而更稀缺了。

祝好！
[你的名字]
```

### 2.2 微信群 — 自然引入

```
各位前辈好，我是[名字]，目前在做一个有趣的项目想请教大家。

我们观察到一个现象：AI很火，但中小微企业不知道怎么用；
而很多资深行业专家经验丰富，但不知道怎么变现。

我们在做"融智桥"——一个让专家和AI人才搭班子的平台。

想请教群里各位大佬：如果有一个平台，给您配一位AI技术搭档，
您只需要贡献行业经验判断，技术实现由AI人才完成，收入按约定
比例自动分账——您会感兴趣吗？

如果方便，麻烦填一下这个问卷（3分钟），帮我们验证一下方向：
[专家端问卷链接]

感谢！🙏
```

```

我们做"融智桥"的初衷就是：AI时代最缺的不是技术人才，是"懂行业、

我们想搭建一个平台，让专家不需要学AI，就能把自己的经验变成
可交付的AI方案。专家诊断问题、审核方案；AI人才负责技术实现。

邀请您花3分钟填份需求调研问卷：[专家端问卷链接]

另外，如果您愿意做一次20分钟的深度访谈，我们可以聊聊平台对
```

## 三、常见异议处理

| 异议 | 回应 |
|------|------|
| "我不懂AI，做不了这个" | 这正是我们的设计——**你不需要懂AI**。你懂行业、懂业务、懂问题在哪。AI人才负责技术实现。你只需要"诊断+审核"。 |
| "我年纪大了，不想折腾" | 不是折腾，是**轻量参与**。每月投入5-15小时即可，线上就能完成。你的经验判断可能值1000元/小时。 |
| "这模式靠谱吗？会不会被骗？" | 平台用**区块链智能合约**自动分账，规则写在代码里，谁也改不了。而且先DEMO验证，满意再正式合作。 |
| "我已经退休了，不想再工作" | 不是"上班"，是**按项目合作**。有合适的项目接，没有就不接。很多专家退休后反而觉得"闲得慌"，这不失为一种充实的方式。 |
| "我看看再说" | 完全理解。您可以先填问卷（3分钟），我们后续有DEMO了第一时间邀请您体验。没有任何承诺，先看看再说。 |
| "我值多少钱？怎么定价？" | 根据调研，资深专家时薪在500-2000元/时。平台建议您先按"项目打包价"试水，后续根据客户反馈调整。我们也会提供定价参考。 |

## 四、种子专家权益设计

| 权益 | 说明 |
|------|------|
| 🔹 首批认证标签 | 平台正式上线后，种子专家享有"创始专家"认证徽章 |
| 🔹 零平台费期 | 前3个项目免平台服务费（100%收入归专家和AI人才） |
| 🔹 优先匹配权 | 新企业需求优先推送给种子专家 |
| 🔹 技能包署名 | 由您参与提炼的技能包，永久署名+销售分成 |
| 🔹 推荐双倍积分 | 种子专家推荐新成员，积分翻倍（4分/人） |
| 🔹 DEMO共建者 | 您的名片和个人介绍将出现在DEMO中，获得首批曝光 |
