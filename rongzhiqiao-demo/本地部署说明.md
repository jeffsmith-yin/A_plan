# 融智桥 DEMO — 本地部署说明

## 环境要求

- Node.js v18+（推荐 v22）
- npm 或 pnpm

## 部署步骤

### 1. 解压

```bash
tar -xzf rongzhiqiao-demo.tar.gz
cd rongzhiqiao-demo
```

### 2. 安装依赖

```bash
npm install
# 或
pnpm install
```

### 3. 启动（选择一种方式）

#### 方式A：直接运行前端（开发模式）

```bash
npm start
```

浏览器自动打开 `http://localhost:3000`，支持热更新。

**注意**：开发模式下 AI 功能走的是 `proxy.js` 同域代理，所以需要同时启动代理服务器。

#### 方式B：使用代理服务器（推荐，支持 AI 功能）

```bash
# 先构建前端
npm run build

# 启动代理服务器（同时提供静态文件和 AI API 代理）
node proxy.js
```

浏览器访问 `http://localhost:3000`

### 4. 配置 AI 功能

1. 前往 [腾讯云混元控制台](https://console.cloud.tencent.com/hunyuan/start) 申请 API Key
2. 在 DEMO 的「文件管理」页面点击「API 设置」
3. 输入 API Key，点击「验证」
4. 验证成功后即可使用 AI 总结和自我介绍生成功能
