# 融智桥 · 运营增长方案（种子招募 + 短视频获客矩阵）

> **文档性质**：运营专家 **Draft（草案）**，供内部执行与迭代，**非投放承诺**。
> **版本**：v0.1　**适用阶段**：DEMO 冷启动 → MVP 预热的种子期（0–3 个月）
> **目标**：① 建起 **30 行业专家（35+）+ 20 AI 人才** 种子库；② 搭建**国内 + 海外短视频/社媒获客矩阵**，为正式产品蓄客。
> **衔接**：推荐裂变接产品积分（新成员 +10 / 推荐 +2 去重）、NDA 接法律模板、DEMO 接闯关原型。

---

## 一、总目标与北极星

| 维度 | 种子期目标（0–3 月） |
|------|----------------------|
| 种子专家 | **30 人**（35+ 经验专家，中国语境） |
| 种子 AI 人才 | **20 人**（Agent / 数据 / 工程化） |
| 内容管线 | 国内 5 平台矩阵 + 海外 3 组渠道稳定周更 |
| 北极星指标 | **有效 DEMO 发起数**（企业代表完成闯关并提交痛点） |
| 次级指标 | 入驻→推荐裂变系数、短视频引流到企微/小程序转化率 |

> 北极星不选「播放量」——播放是虚荣指标，DEMO 发起才是真需求信号。

## 二、种子专家 + AI 人才招募方案

### 2.1 专家画像与筛选标准
- **核心供给侧**：中国 **35 岁以上**、经验丰富但再就业困难的行业专家（≠ 退休专家）；行业年限 **≥15 年**，有可口述、可交付的方法论。
- **行业覆盖（建议首批）**：制造业（排产/质量）、零售/连锁、餐饮、外贸/跨境、物流、财税、HR、法务。
- **可交付能力**：能定义「规则/约束」、有代表案例、愿意参与 DEMO 闯关与企业对话。
- **出镜意愿**：短视频矩阵需要出镜口播，标注「愿出镜 / 仅文案 / 仅方案」。
- **红线排除**：纯割韭菜卖课、无法交付、与「骗局同构」的立门派套路（见竞品分析警示）。

### 2.2 AI 人才画像
- Agent 编排（Coze/Dify/自建）、数据工程、模型微调、前端 H5/小程序；
- 能承接「专家定规则 → AI 配 Agent → 企业验收」的工程化；
- 熟悉 DEMO 闯关链路，能输出脱敏草案与预警。

### 2.3 来源渠道
| 渠道 | 适用 | 打法 |
|------|------|------|
| 脉脉 | 职场/高管 | 关键词搜「前总监/合伙人/总」，私信邀约 |
| 知乎 | 行业答主 | 回答区识别高质量答主，评论区钩子引流 |
| 行业微信群 / 企微 | 私域 | 通过现有专家内推进群，群内招募 |
| 退休/再就业专家组织 | 35+ 供给侧 | 协会合作、公益讲座转化 |
| EMBA / 总裁班 / 行业协会 | 高信任 | 联合沙龙、案例共创 |
| 技术社区（GitHub/掘金/CSDN） | AI 人才 | 项目共建、内测席位吸引 |

### 2.4 推荐裂变（接产品积分）
- 与产品规则一致：**新成员 +10 分、推荐 +2 分、去重不计**（见 Uni-app MineView）。
- 种子期加码：**推荐 1 位通过审核的专家 → 赠 DEMO 内测席位 + 联合案例署名权**。
- 入口：名片页「推荐裂变」按钮，生成带标识的邀请（微信/手机），后端去重。

### 2.5 入驻与激活（aha 时刻）
1. 填名片（姓名/电话/微信/职务/个人介绍）；
2. **在线签 NDA**（数字水印，见法律模板）；
3. **完成 1 次 DEMO 闯关**（制造业排产 5 步）作为激活门槛；
4. 进入种子库标签体系，分配「可交付方向」。

### 2.6 激励与留存
- 早期入驻权益：永久授权折扣、专属数据看板（专家维度）、联合案例署名；
- 里程碑激励：首批 30+20 达成后，举办「融智桥首批专家授牌」直播，强化归属感；
- 反流失：每月 1 次「专家×AI 人才」对齐会，持续明确各角色介入时间表（见 PRD）。

### 2.7 招募节奏（8 周里程碑）
| 周次 | 动作 | 目标 |
|------|------|------|
| W1–2 | 锁定画像、建渠道清单、设计邀请话术 | 渠道就绪 |
| W3–5 | 首批邀约 + 内推 | 15 专家 + 10 AI |
| W6–8 | 补齐 + 授牌直播 | 30 专家 + 20 AI |

### 2.8 种子库管理（轻量 CRM）
- 标签：行业 / 可交付方向 / 出镜意愿 / 时区 / 状态（待触达→已入驻→已激活→已交付）；
- 维护：`outreach/` 目录下的招募表（已存在骨架），周更。

## 三、短视频科普获客矩阵

### 3.1 内容栏目（统一供给，按平台裁剪）
- **栏目 A《我踩过的坑》**：35+ 专家口播，讲真实经营痛点（强信任、强人设）。
- **栏目 B《AI×行业 1 分钟》**：把复杂问题拆成 AI 能解的题（科普、涨粉）。
- **栏目 C《DEMO 走查》**：制造业排产闯关录屏（功能≤50% / 核心≤20% 徽标，演示数据）。
- **栏目 D《企业主对话》**：专家 + AI 人才 + 企业主三方对谈，展示协作形态。

### 3.4 海外冷启动建议
- **先攻 LinkedIn + YouTube + TikTok** 三件套（B2B 背书 + 深度 + 破圈）；
- 即时通讯类（WhatsApp/Telegram/Discord）做**私域承接**，不做首发获客；
- region 分包：东亚（LINE/Kakao/Zalo）、欧美（FB/IG/TikTok/LinkedIn/X）、东欧（VK/Telegram）；
- 内容本地化：专家口播配字幕，DEMO 录屏做英文/当地语旁白。

### 3.5 冷启动打法（0–30 天）
1. 前 **20 条**做内容测试（A/B/C/D 栏目各 5），看完播与引流；
2. 选 2 条数据最好栏目在抖音/视频号**小额投流**（豆荚/boost）；
3. 评论区统一话术：「演示数据，私信领 DEMO 内测」→ 引流企微/小程序；
4. 每周复盘：把引流转化率最高的平台加频，最差降级。

### 3.6 KPI（种子期）
| 指标 | 目标 |
|------|------|
| 短视频总发布 | ≥ 60 条 / 月（国内矩阵） |
| 企微/小程序新增 | ≥ 300 / 月 |
| DEMO 发起数 | ≥ 50 / 月（北极星） |
| 推荐裂变系数 | ≥ 1.15（人均带来 0.15 新成员） |

### 3.7 合规注意
- 所有 DEMO 内容标注「**演示数据**」，不承诺效果、不暗示真实交易；
- 专家出镜须实名授权，个人介绍遵循 PIPL 最小必要；
- 海外发布注意当地数据与广告合规（如 GDPR 区域、平台广告政策）。

## 四、与产品 / 技术衔接
- **推荐裂变** → Uni-app `MineView` 积分（+10/+2 去重），ChainAdapter 账本可扩展推荐记录；
- **NDA** → 法律模板在线签署，数字水印 + 到期撤销 API Key；
- **DEMO 闯关** → `HomeView` 制造业 5 步，作为专家激活与获客钩子；
- **数据看板** → `DashboardView` 10 类指标，运营可监测招募/获客进展。
